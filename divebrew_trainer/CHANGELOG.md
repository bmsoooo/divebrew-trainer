# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1+1] - 2026-07-10

### Added
- Added user current location integration using `geolocator` for Marine Conditions.
- Expanded domestic marine dive sites to a total of 16 locations (Goseong, Sokcho, Yangyang, Gangneung, Donghae, Uljin, Yeongdeok, Pohang, Ulleungdo, Busan, Geoje, Tongyeong, Namhae, Yeosu, Jeju, Seogwipo).

### Changed
- Replaced default Web App Icons with the DiveBrew specific icon (`favicon.png`, `Icon-192.png`, `Icon-512.png`, `Icon-maskable-192.png`, `Icon-maskable-512.png`).
